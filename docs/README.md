## Como preencher (resumo)

1. Escolher **um** módulo.
2. Ler o código real correspondente.
3. Preencher só as seções do módulo, seguindo o `_MODELO.md`.
4. Atualizar o **Status** e a data no cabeçalho.
5. Parar e aguardar o próximo módulo.
